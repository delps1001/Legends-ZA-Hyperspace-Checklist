PHASE 0

Pre-DLC Prep

~2–3 hr
Don't skip this. Front-loading prep saves ~8 hours over the full run.

// TRAINING & GEAR

  • Train at least 3 Pokémon to Lv. 100 (Hyperspace mons spawn Lv. 100–200; no EXP earned in-zone)
  • Max Blue Canari Plush to Lv. 3 at Colorful Screw Exchange (8 screws) — biggest catch-rate boost in game
  • Max Pink Canari Plush to Lv. 3 (3× Mega Shard yield, needed for Magearna grind)
  • Clear base-game SM 133 "Farfetch'd Ambush" for the False Swipe TM
    • Talk to Bouche at Jaune Pokémon Center
    • Fight three Farfetch'd trainers at Rouge Sector 7 at night
  • Put False Swipe on a fast Pokémon (Gallade or Sirfetch'd recommended)

// ITEM STOCKPILE

  • 50+ Ultra Balls
  • 20+ Quick Balls (first-turn throws)
  • 20+ Dusk Balls (night catches)
  • 30-stack of every common berry from Lumiose berry stalls

// MYSTERY GIFTS TO REDEEM

  • Diancite (live since Nov 6, 2025)
  • Mewtwonite X & Y (live since Nov 19, 2025)
  • Garchompite Z (live since Feb 27, 2026)
  • Luxurious Poké Ball Set (email code from DLC purchase — 12 Apricorn balls)

PHASE 5

Post-Credits Cleanup

~10–18 hr
RNG-heavy. The bottleneck of the entire run.

// UNLOCK SPECIAL SCAN

  • Visit Rust Syndicate Office → take SM 188 "Start Special Scanning!"
  • Talk to Philippe (Special Scan = 25,000 Survey Points per use)
Don't scan yet — finish named missions first while farming organically.

// NAMED-MISSION LEGENDARIES

  • SM 191 "Collecting Four Drives" → Genesect ✓
  • SM 192 "The Stealthy Shadow" → Marshadow ✓
  • SM 193 "Dreams of Meltan"
    • Catch Meltan ✓
    • Receive gift Melmetal ✓ (answer "yes" to evolve)
  • SM 194 "Volcanion Unleashed" → Volcanion ✓
  • SM 196 "The Djinn Unbound" → battle Hoopa Unbound at Bleu Sector 6 rooftop, awards Prison Bottle → Hoopa ✓
  • SM 195 "Restarting Magearna" → give Jett 999 Mega Shards → Magearna ✓ + Magearnite

// PRIMALS + RAYQUAZA (HM 12–14)

⚠️ Backup save before crafting each Special Donut (title screen → Up + X + B). Recipes are one-shot per playthrough. Reroll for Sparkling Power Lv. 3 if you want shinies.
  • Start HM 12 Pt. 1 "The Greatest Gift" (70,000 pts) — pick Omega or Alpha first
  • Craft Omega Old-Fashioned Donut → ruby distortion at Jaune Sector 11 → catch Primal Groudon ✓ + Red Orb (HM 13)
  • Craft Alpha Old-Fashioned Donut → WZ5 distortion → catch Primal Kyogre ✓ + Blue Orb (HM 14)
  • Complete HM 12 Pt. 2 (100,000 pts)
  • Craft Delta Old-Fashioned Donut (sour 340 / fresh 390) → Hyperspace Sky Pillar atop Hotel Z
  • Defeat & catch Lv. 200 Mega Rayquaza ✓ (guaranteed catch)

// SPECIAL SCAN GRIND (THE BIG BOTTLENECK)

  • Pre-req: take SM 189 "Siblings of the Sky" (unlocks Latias/Latios in scan pool + their Mega Stones)
  • Scan + catch Latias ✓
  • Scan + catch Latios ✓
  • Scan + catch Cobalion ✓
  • Scan + catch Terrakion ✓
  • Scan + catch Virizion ✓
  • Scan + catch Keldeo ✓
  • Scan + catch Meloetta ✓
📊 Each scan costs 25,000 pts (~3 Battle Zone clears = 30–45 min). Only "Unknown" 5★ zones contain these. Community average: 15–25 scans for all 7. None are shiny-locked — backup-save SR if you want shinies.
PHASE 6

EX Missions + Zeraora

~2–3 hr

// EX MISSIONS

  • EX 1 "Shine Bright Like a Gemstone" (Diancite required)
    • Visit Looker Bureau → follow Mimi the Espurr
    • Catch Lv. 70 Diancie protected by 2× Lv. 66 Carbink (not in Hyperspace dex but unlocks EX 3)
  • EX 2 "Project M" (Mewtwonite X/Y required)
    • Lysandre Café elevator → Lysandre Labs B3F
    • Catch Lv. 70 Mewtwo (not in Hyperspace dex but unlocks EX 3)

// EX 3: ZERAORA

  • Buy Canari Bread (₽1,300) from Tarragon at Racine Construction
  • Trade to boy on Vert Sector 3 rooftop (Holovator near Café Cyclone) → receive Popping Candy
  • Give Popping Candy to Ansha → unlocks Plasma-Glazed Donut recipe (sour 400 / bitter 280 / spicy 200)
  • Answer Emma's call → enter yellow distortion at Gare de Lumiose
  • Defeat Lv. 200 Rogue Mega Zeraora (Ground-type counter recommended)
  • Catch Zeraora ✓ + Zeraorite

// GARCHOMPITE Z EVENT

  • Enter special distortion on map (live event, no end date)
  • Defeat Mega Garchomp Z → receive Garchompite Z (not a dex entry, completes Mega collection)
